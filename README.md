## 🔍 使い方ガイド

### 新規記事の作成
1. 該当カテゴリフォルダを開く
2. 「Add file」→「Create new file」をクリック
3. テンプレートを参考に記事を作成
4. Commit changesで保存

### 記事の編集
1. 編集したい記事を開く
2. 鉛筆アイコン（Edit）をクリック
3. 編集後、Commit changesで保存

### 検索
- **上部の検索バー**で記事内容を検索可能
- フォルダ名やファイル名でも検索できます

## 💡 Tips

- 📝 Markdownで記述（書式は簡単！）
- 🔗 `[[リンク]]`で他の記事と連携
- 🖼️ 画像は`assets/images/`フォルダに保存
- ⚡ GitHubは自動でバックアップ・履歴管理
